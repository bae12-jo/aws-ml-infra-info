Need a Custom AMI?
🔧 None of these fit? Build your own.

Use any DLAMI as a ParentImage and layer your own packages on top via pcluster build-image. Works for both ParallelCluster and HyperPod. Recommended starting point: DLAMI Base with Single CUDA (Ubuntu 22.04) — bare OS + CUDA driver, nothing extra. 

Build:
  ParentImage: ami-<dlami-base-single-cuda-ubuntu2204>
  BuildInstanceType:
    - g5.xlarge
  Components:
    - Type: script
      Value: s3://<bucket>/scripts/install-frameworks.sh

# pcluster build-image --image-id my-custom-ami \
#   --image-configuration build-config.yaml \
#   --region us-east-1
